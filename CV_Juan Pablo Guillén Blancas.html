Proyectos Relevantes (Académicos)

PCB para Prototipo de Dron de Rescate

Diseñé e implementé una PCB multicapa para un prototipo de dron de rescate, integrando un microcontrolador central con cámara térmica para detección de calor, sensores ultrasónicos para navegación y sistemas de alerta visual y acústica mediante LEDs y buzzer. Incluyó módulo de comunicación inalámbrica y regulador DC-DC para estabilizar la alimentación desde una batería Li-Po de 5 V. Validé la PCB mediante pruebas de continuidad y ensamblaje exitoso en un prototipo funcional.

Desarrollé el programa en C++ con Arduino IDE para el microcontrolador ESP32, procesando y mostrando señales térmicas, obteniendo coordenadas GPS (módulo Neo-6M), midiendo distancias con sensores ultrasónicos y transmitiendo datos mediante interfaz inalámbrica.

Robot Cartesiano de 3 Ejes — Proyecto Final de Licenciatura

Desarrollé un robot cartesiano a escala como proyecto final de Ingeniería Mecatrónica, incluyendo diseño completo en SolidWorks y fabricación de piezas mediante impresión 3D. Implementé una PCB de control personalizada con microcontrolador ESP32, drivers de potencia para motores DC/BLDC con encoders y sensores de fin de carrera. Incluyó circuito de respaldo para conservar datos de encoders ante pérdida de energía y topología de distribución eléctrica estable.

El firmware fue desarrollado en C++ con Arduino IDE, permitiendo lectura continua de posiciones, ejecución de rutinas de homing, movimientos automáticos, y sistema de seguridad ante excedentes de recorrido. Incluyó documentación técnica completa (esquemas, PCB, BOM, manual de cableado).
